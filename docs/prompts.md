# Prompt Catalog

Store prompt templates here instead of scattering them inside widgets.

## bullet_rewrite

Rewrite a resume bullet to improve clarity, action, and measurable impact while preserving factual accuracy.

## summary_rewrite

Rewrite a professional summary for stronger positioning against a target role without inventing claims.

## job_tailoring

Tailor resume language to a target job description using only information already present in the resume.

## skills_gap_explanation

Explain which keywords or skills appear missing from the resume and what can be reframed from existing experience.
